## 项目简介

快点HAM工具箱（Hello，CQ）是一个为业余无线电爱好者（HAM）提供在线工具和服务的综合平台。本项目旨在为HAM社区提供便捷的工具集合，帮助爱好者更好地进行无线电通联活动。

## 主要功能

- 📡 实时时钟显示（北京时间/UTC时间）
- 🌍 传播预测工具
  - UV传播预测
  - 短波传播预测
  - 空间环境监测
  - 实时太阳观测
  - 电离层TEC现报
- 🗺️ 实用工具集
  - 业余无线电中继查询
  - 梅登黑德网格定位
  - CW练习与翻译
  - 卫星过境查询
  - NOAA解码
  - SDR网页客户端
  - 短波广播查询
- 📚 学习资源
  - 业余无线电基础知识
  - 在线考试模拟
  - HAM相关视频
  - 操作技术指南
